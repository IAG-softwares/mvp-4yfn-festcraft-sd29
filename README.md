# FestCraft

Creative platform for dance festival organizers to centralize ideation, track innovative concepts, and manage diverse festival formats from inception to execution.

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

```
├── frontend/          # React + Vite frontend
│   ├── src/

│   │   └── api/      # API client
│   └── ...
├── backend/           # Express.js backend
│   ├── routes/       # API routes
│   ├── database.js   # SQLite database
│   └── server.js     # Express server
└── scripts/          # Helper scripts
```

## Features

Creative idea management system with categorization by dance style and format, innovation tracking with feasibility scoring, collaborative concept development tools, festival format templates and customization options, budget and resource estimation for each concept, timeline and milestone tracking, team collaboration features, visual mood boarding for concepts, venue and capacity planning tools, marketing angle documentation, and comprehensive analytics dashboard showing idea progression and success rates.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
